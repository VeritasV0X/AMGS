# AMGS App Store Submission Packet

Prepared on April 25, 2026 for AMGS Final.

This packet contains draft documents and copy for App Store submission. It is not legal advice. Review all documents against the actual final app build before submitting.

## Most important assumption

This packet assumes the app is primarily a local SwiftUI/SwiftData app with optional live AI API calls and local fallback answers. If you add iCloud sync, analytics, crash reporting, accounts, ads, subscriptions, social features, or public user-generated content, update these documents before submission.

## Public pages you need to host

Apple requires a Privacy Policy URL for apps. You should host:

- `Privacy_Policy.html` at your privacy URL.
- `Support_Page.html` at your support URL.
- `Terms_of_Use.html` if using custom terms.
